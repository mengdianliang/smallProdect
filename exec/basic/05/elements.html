        1.<a>默认样式：
            a:-webkit-any-link {
                color: -webkit-link;
                cursor: auto;
                text-decoration: underline;
            }
        2.<span>默认样式：无，一般用来区分样式
        3.<div>默认样式：
            div {
                display: block;
            }
        4.<img>默认样式：无
        5.h1~h6默认样式：
             {
                display: block;
                font-size: 2em;
                -webkit-margin-before: em;
                -webkit-margin-after: em;
                -webkit-margin-start: 0px;
                -webkit-margin-end: 0px;
                font-weight: bold;
            }
        6.<p>默认样式：
            p {
                display: block;
                -webkit-margin-before: 1em;
                -webkit-margin-after: 1em;
                -webkit-margin-start: 0px;
                -webkit-margin-end: 0px;
            }
        7.<strong>默认样式：
            跟b元素一样起到加粗的作用，但是strong更强调
            strong, b {
                font-weight: bold;
            }
        8.<em>默认样式：
            跟i元素一样起到倾斜的作用，但是em更强调，其他的也有自己的语义
            i, cite, em, var, address, dfn {
                font-style: italic;
            }
        9.ul/ol li默认样式：
            ul, menu, dir {
                display: block;
                list-style-type: disc;
                -webkit-margin-before: 1em;
                -webkit-margin-after: 1em;
                -webkit-margin-start: 0px;
                -webkit-margin-end: 0px;
                -webkit-padding-start: 40px;
            }
            ol {
                display: block;
                list-style-type: decimal;
                -webkit-margin-before: 1em;
                -webkit-margin-after: 1em;
                -webkit-margin-start: 0px;
                -webkit-margin-end: 0px;
                -webkit-padding-start: 40px;
            }
            li {
                display: list-item;
                text-align: -webkit-match-parent;
            }
        10.dl,dt,dd默认样式：
            dl {
                display: block;
                -webkit-margin-before: 1em;
                -webkit-margin-after: 1em;
                -webkit-margin-start: 0px;
                -webkit-margin-end: 0px;
            }
            dt {
                display: block;
            }
            dd {
                display: block;
                -webkit-margin-start: 40px;
            }
        11.<hr>默认样式：
            hr {
                display: block;
                -webkit-margin-before: 0.5em;
                -webkit-margin-after: 0.5em;
                -webkit-margin-start: auto;
                -webkit-margin-end: auto;
                border-style: inset;
                border-width: 1px;
            }
        12.<br>默认样式：无
        13.<mark>默认样式：
            mark {
                background-color: yellow;
                color: black;
            }
        

         1.结构化标签的默认样式：
            article, aside, footer, header, hgroup, main, nav, section {
                display: block;
            }
        2.<time>默认样式：无
        

        1.行内元素：a,span,br,strong,em,time
        2.块级元素：div,hr,h1~h6,p,ul,ol,li,dl,dt,dd,header,footer,section,article,nav,aside
        3.既不是块级也不是行内元素的：img 四要素：src width/height alt title
        一般兼容问题要加上：
            img{
                vertical-align:top;
                border:none;
            }
